The following endpoint retrieves the settlement based on ID

<url get = ""><div class = "ib">/settlements/:id</div></url>

</div>

<div class = 'full-side'>

<div class = 'left-side'>

**Query parameters**

Following are the parameters to be sent in the request body:

| Field |	is required? |	Description |
| ------- | ------------- | ------------- |
| _id_ |	`mandatory` |	`string` Unique identifier of the settlement ID |

</div>
