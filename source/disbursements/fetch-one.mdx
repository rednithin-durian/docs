The following endpoint retrieves the details of the disbursement created

<url get = ""><div class = "ib">/disbursements/:id</div></url>

</div>

<div class = 'full-side'>

<div class = 'left-side'>

**Path parameters**

| Field |	is required? |	Description |
| ----- | ---------------- | ---------- |
| _id_ |	`mandatory` |	`string` Unique identifier of the disbursement to be retrieved. |

</div>

<div class = 'right-code'>

**Example**

<MultiCodeBlock>

```bash
curl -u [Base64({Your_Server_Key}:)] \
-X GET https://api.durianpay.id/v1/disbursements/dis_XXXXXXX \
-H "content-type: application/json"'
```

```python
import http.client

conn = http.client.HTTPSConnection("api.durianpay.id")

headers = {
    'content-type': "application/json",
    'authorization': "[Base64({Your_Server_Key}:)]",
    }

conn.request("GET", "/v1/disbursements/dis_iQsHCpYoxo5744", headers=headers)

res = conn.getresponse()
data = res.read()

print(data.decode("utf-8"))
```

```java
OkHttpClient client = new OkHttpClient();

  .addHeader("authorization", "[Base64({Your_Server_Key}:)]")
  .build();

```
