The following endpoint retrieves the balance in the durianpay account.

<url get = ""><div class = "ib">/disbursements/topup/balance</div></url>

</div>

<div class = 'right-code ml-60'>

**Example**

<MultiCodeBlock>

```bash
curl -u [Base64({Your_Server_Key}:)] \
-X GET https://api.durianpay.id/v1/disbursements/topup/balance \
-H "content-type: application/json"'
```

```python
import requests

url = "https://api.durianpay.id/v1/disbursements/topup/balance"

headers = {
    'content-type': "application/json",
    'authorization': "[Base64({Your_Server_Key}:)]",
    }

response = requests.request("GET", url, headers=headers)

print(response.text)
```

```java
OkHttpClient client = new OkHttpClient();

Request request = new Request.Builder()
  .url("https://api.durianpay.id/v1/disbursements/topup/balance")
  .get()
  .addHeader("content-type", "application/json")
  .addHeader("authorization", "[Base64({Your_Server_Key}:)]")
  .build();

Response response = client.newCall(request).execute();
```

```js
var request = require("request");

var options = { method: 'GET',
  url: 'https://api.durianpay.id/v1/disbursements/topup/balance',
  headers: 
   {  authorization: '[Base64({Your_Server_Key}:)]',
     'content-type': 'application/json' } };

  console.log(body);

```

```ruby
require 'uri'
require 'net/http'

url = URI("https://api.durianpay.id/v1/disbursements/topup/balance")

http = Net::HTTP.new(url.host, url.port)
http.use_ssl = true
http.verify_mode = OpenSSL::SSL::VERIFY_NONE

request = Net::HTTP::Get.new(url)
request["content-type"] = 'application/json'
request["authorization"] = '[Base64({Your_Server_Key}:)]'

response = http.request(request)
puts response.read_body
```

</MultiCodeBlock>

#### Response Code : 200 
Here is an example of success response

```js
{
    "data": {
        "balance": 949859471313
    }
}
```

#### Error Response Code : 401
Here is an example of error response for invalid request

```js
{
    "error": "invalid Authorization header in request"
}
```
