The following endpoint retrieves the details of all Promos created

<url get = ""><div class = "ib">/merchants/promos/:id</div></url>

</div>

<div class = 'full-side'>

<div class = 'left-side'>

**Path parameters**

| Field |	is required? |	Description |
| ------- | --------------- | ------------ |
| _id_ |	`mandatory` |	`integer` Unique identifier of the Promo to be retrieved |

</div>

<div class = 'right-code'>

**Example**

<MultiCodeBlock>

```bash
curl -X GET \
  https://api.durianpay.id/v1/merchants/promos/prm_vnvN2Rp1tU7753 \
  -H 'authorization: [Base64({Your_Server_Key}:)]' \
  -H 'content-type: application/json' \
```

```python
import http.client

conn = http.client.HTTPSConnection("api.durianpay.id")

headers = {
    'content-type': "application/json",
    'authorization': "[Base64({Your_Server_Key}:)]",
    }

conn.request("GET", "/v1/merchants/promos/prm_vnvN2Rp1tU7753", headers=headers)

res = conn.getresponse()
data = res.read()

print(data.decode("utf-8"))
```
