The following endpoint deletes a promo

<url delete = ""><div class = "ib">/merchants/promos/:id</div></url>

</div>

<div class = 'full-side'>

<div class = 'left-side'>

**Path parameters**

| Field |	is required? |	Description |
| ----- | --------------- | ------------- |
| _id_ |	`mandatory` |	`integer` Unique identifier of the Promo to be deleted |

</div>

**Example**

```bash
curl -X DELETE \
  https://api.durianpay.id/v1/merchants/promos/prm_vnvN2Rp1tU7753 \
  -H 'authorization: [Base64({Your_Server_Key}:)]' \
  -H 'content-type: application/json' \
```

```python
import http.client

conn = http.client.HTTPSConnection("api.durianpay.id")

headers = {
    'content-type': "application/json",
    'authorization': "[Base64({Your_Server_Key}:)]",
    }

conn.request("DELETE", "/v1/merchants/promos/prm_vnvN2Rp1tU7753", headers=headers)

res = conn.getresponse()
data = res.read()

print(data.decode("utf-8"))
```

<div class = 'right-code ml-60'>

#### Response Code : 200 
Here is an example of success response

```js
{
    "message": "Promo deleted successfully"
}
```

#### Error Response Code : 401
Here is an example error response for unauthorized access

```js
{
    "error": "invalid Authorization header in request",
    "error_code": "DPAY_UNAUTHORIZED_ACCESS"
}
```

#### Error Response Code : 500
Here is an example error response for internal server error

```js
{
    "error": "error deleting promo",
    "error_code": "DPAY_INTERNAL_ERROR"
}
```
