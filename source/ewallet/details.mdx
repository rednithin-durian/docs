## Details EWallet Account

The following endpoint gets Gopay Account Details:

<url get = ""><div class = "ib">/ewallet/account?mobile="88888888888"?wallet_type="GOPAY"</div></url>

</div>

<div class = 'full-side'>

<div class = 'left-side'>

**Request parameters**

Following are the parameters to be sent in the url query param :

| Field |	is required? |	Description |
| ------ | ---------------- | ------------- |
| _mobile_ | 	`mandatory` |	`string` The mobile you use for e-wallet account |
| _wallet_type_	 | `mandatory` |	`string` The wallet you use ex GOPAY |

</div>

#### Error Response Code : 400

Here is an example of error response for invalid request

```js
{
    "error": "Mobile number is missing in request",
    "error_code": "DPAY_INVALID_REQUEST"
}
```
